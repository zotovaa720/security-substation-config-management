# Проектная baseline комплексной системы безопасности энергетической подстанции

Данный Git-репозиторий содержит проектную baseline и комплект
эксплуатационной документации комплексной системы безопасности
энергетической подстанции, включающей подсистемы видеонаблюдения
и контроля и управления доступом (СКУД).

Репозиторий используется как инструмент управления конфигурацией
в соответствии с принципами ISO/IEC 10007 и практиками ITIL.

---

## Цель репозитория

Целью создания репозитория является:
- формирование единой проектной baseline системы безопасности;
- фиксация конфигурационных единиц (Configuration Items, CI);
- обеспечение трассируемости архитектурных и эксплуатационных решений;
- демонстрация применения Git как средства управления конфигурацией
  в рамках учебного проекта.

---

## Краткое описание системы

Объектом проектирования является энергетическая подстанция
с круглосуточным режимом работы и ограниченным составом персонала
(20 инженеров).

Система безопасности предназначена для:
- предотвращения несанкционированного физического доступа;
- круглосуточного видеонаблюдения за критическими зонами;
- мониторинга перегрева оборудования в распределительном устройстве;
- формирования аудируемой доказательной базы событий доступа;
- обеспечения автономной работы при аварийных ситуациях.

Архитектура системы — локальная централизованная
с элементами автономии.
